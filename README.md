# Sistema Académico

## Descripción
Este proyecto consiste en un sistema académico desarrollado como aplicación de escritorio, el cual permite gestionar la información de estudiantes, maestros, materias y áreas técnicas.

El sistema incluye funcionalidades como:
- Registro de estudiantes
- Registro de maestros
- Registro de materias
- Registro de áreas técnicas
- Relación de estudiantes con sus áreas
- Visualización de datos en tablas
- Generación de reportes académicos

El objetivo del sistema es facilitar la organización y el control de la información académica de manera rápida y estructurada.

## Tecnologías Utilizadas
- Lenguaje(s): C#
- Herramientas / Framework: Windows Forms y Biblioteca de clases
- Base de datos: SQL Server
- Entorno o IDE: Visual Studio



## Uso o Ejecución
1. Abrir el proyecto en Visual Studio.
2. Ejecutar el sistema presionando F5.
3. Registrar estudiantes, maestros, materias y áreas técnicas.
4. Visualizar los reportes generados por el sistema.
